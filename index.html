Chompie

A high level virtual machine source on github

Chompie simulates a very simple machine. This machine can get and set variables and grouped variables (objects and arrays) It can do if and goto.

It takes an array of (nested) commands and evaluates them.

Usage

chompie relies on the extremely simple poor_module. It also uses peppermint-expressions as an easy way to create the arrays that chompie uses.

Browser

<script src="poor_modules/poor-module.js"></script>
<script src="peppermint-expressions.js"></script>
<script src="chompie.js"></script>
<script>
var chompie = poorModule("chompie")
chompie('say: "hello world!"')
</script>

node.js

require("./poor_modules/poor_module.js")
require("./peppermint-expressions.js")
require("./chompie.js")
var chompie = poorModule("chompie")
chompie('say: "hello world!"')
